📌 Project Overview

This project detects whether workers are wearing proper safety equipment like:

Helmet
Safety Vest
Gloves
Goggles

It uses YOLOv8 for real-time detection via webcam.

🚀 Features
Real-time PPE detection
Email alert system 🚨
Multiple PPE detection (Helmet, Vest, Gloves, Goggles)
Violation detection (No Helmet, No Vest)
🛠️ Technologies Used
Python
OpenCV
YOLOv8 (Ultralytics)
SMTP (Email Alerts)
▶️ How to Run
pip install -r requirements.txt
python main.py
📸 Output
Live camera feed
Bounding boxes for detected PPE
Alert if safety rules are violated
📧 Alerts

System sends email if:

Person detected without helmet
